

1. **What is Linux, and how is it different from Unix?**
   Explain the architecture, open-source nature, kernel, and similarities/differences with Unix systems.

2. **What are the basic Linux file permissions?**
   Describe:

   * Read (`r`)
   * Write (`w`)
   * Execute (`x`)
     Also explain permission groups:
   * User
   * Group
   * Others

3. **What is the difference between `grep`, `find`, and `locate` commands?**

   * `grep` → searches text inside files
   * `find` → searches files/directories
   * `locate` → quickly finds files using a database

4. **How do you check system resource usage in Linux?**
   Mention commands such as:

   * `top`
   * `htop`
   * `free`
   * `df`
   * `du`
   * `vmstat`

5. **What is the purpose of the `/etc` directory?**
   Explain that it stores system-wide configuration files and scripts.

6. **What is a shell in Linux?**
   Define shell as a command-line interpreter and explain common shells:

   * Bash
   * Zsh
   * Korn Shell

7. **Explain the Linux boot process.**
   Key stages:

   * BIOS/UEFI
   * Bootloader (GRUB)
   * Kernel loading
   * Init/Systemd
   * Services startup

8. **What are soft links and hard links?**
   Compare:

   * Hard link → direct reference to inode
   * Soft link (symbolic link) → shortcut/path reference

9. **How do you manage processes in Linux?**
   Important commands:

   * `ps`
   * `top`
   * `kill`
   * `pkill`
   * `jobs`
   * `nohup`
   * `systemctl`

10. **What is the difference between `cron` and `at` commands?**

* `cron` → recurring scheduled tasks
* `at` → one-time scheduled tasks
